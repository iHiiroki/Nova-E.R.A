# Nova E.R.A
Loja de Eletrônicos - Projeto

Este relatório apresenta o desenvolvimento completo da aplicação web Nova E.R.A.
— uma loja virtual de eletrônicos criada como projeto prático da disciplina de
Orientação a Objetos. O sistema foi desenvolvido em Python e React, adotando a
arquitetura de Single Page Application (SPA) com roteamento baseado em hash. A
aplicação contempla 34 produtos organizados em 8 categorias, sistema de
autenticação com usuários demonstração, carrinho de compras com persistência via
localStorage, além de funcionalidades como busca, filtros, ordenação e simulação de
compra. O back-end, implementado em Python puro com o módulo http.server, serve
a aplicação completa como um único arquivo HTML com JavaScript embutido,
eliminando dependências externas de instalação. O projeto comprova que é possível
construir sistemas web funcionais e visualmente modernos utilizando apenas
ferramentas nativas das linguagens, consolidando conceitos fundamentais de
desenvolvimento web e orientação a objetos.
